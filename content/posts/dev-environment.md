## 编辑器：VS Code

VS Code 依然是我最喜欢的编辑器。以下是我每天都会用的插件：

### 必装插件

- **ESLint** — 代码检查
- **Prettier** — 代码格式化
- **GitHub Copilot** — AI 编程助手
- **Tailwind CSS IntelliSense** — Tailwind 自动补全

### 主题和字体

我使用 `JetBrains Mono` 字体，配合暖色主题保护眼睛。

## 终端：Windows Terminal

Windows Terminal 配合 WSL2，兼顾 Windows 生态和 Linux 开发体验。

## 效率工具

- **Git** — 版本控制，命令行 + VS Code 集成
- **pnpm** — 快速的包管理器
- **Vercel** — 前端项目一键部署

## 小结

好的工具配置能让开发效率翻倍。花一个小时配置好环境，未来能省下几百个小时。
